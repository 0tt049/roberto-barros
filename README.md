# Roberto Barros
A Ruby Gem that randomly generates quotes from the great Roberto Barros.

## Installing
```bash
gem install roberto_barros
```

## Usage
```ruby
require 'roberto_barros'

RobertoBarros.quote #=> Está um ótimo dia para programar.
```
